# 💰 Student Expense Tracker (Python Project)

## 📌 Project Overview
The Student Expense Tracker is a simple Python-based CLI application that helps users track their daily expenses, categorize spending, and generate summaries. It stores data locally using JSON files so that expenses persist even after closing the program.

## 🎯 Features
- Add new expenses (amount, category, date, description)
- View all expenses
- Search expenses by keyword
- Delete expenses
- View total spending
- Category-wise summary
- Monthly spending summary
- Persistent data storage using JSON

## 🛠️ Technologies Used
- Python 3
- JSON (for data storage)
- File Handling
- CLI (Command Line Interface)
